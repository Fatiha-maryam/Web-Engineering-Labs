# Web Engineering Labs

## Overview
This repository contains a collection of HTML lab tasks and exercises completed as part of my **Web Engineering** course. Each lab focuses on different aspects of web development, including structure, forms, tables, lists, and other HTML elements.  

The goal of this repository is to showcase practical implementation of HTML concepts learned in class.

---

## Repository Structure


- **HTML/** – Contains all HTML-related lab tasks.
- **Lab1, Lab2, Lab3** – Subfolders for individual labs, each containing multiple HTML files.

---

## Highlights
- Well-structured folder organization for easy navigation
- Hands-on implementation of key HTML topics
- Demonstrates progression of skills throughout the course

---

## Usage
You can open any HTML file in a browser to view the output of the lab tasks.  

Future updates will include CSS and JavaScript labs as part of a complete web development portfolio.
